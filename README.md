# Hangman_Game

## 📌 Project Description
This is a simple text-based Hangman Game developed in Python as part of the CodeAlpha Python Programming Internship. The player has to guess the hidden word one letter at a time before running out of chances.

## ✨ Features
- Randomly selects a word
- Maximum of 6 incorrect guesses
- Displays guessed letters
- Checks for win or loss
- Console-based gameplay

## 🛠️ Technologies Used
- Python 3

## 📚 Concepts Used
- random module
- while loop
- if-else statements
- Lists
- Strings
- User input/output

## ▶️ How to Run
1. Install Python 3.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run:

```bash
python hangman.py
```
